Descripción detallada

Factoría de componentes.

Extiende la plantilla BaseSubsystems::CFactory para IComponent haciendo de ella un singleton por lo que solo habrá una instancia de esta clase.

Es usada para almacenar componentes en Logic::CEntityFactory donde todo el registro de componentes se realiza utilizando macros del preprocesador declarados en Logic::CComponent.
