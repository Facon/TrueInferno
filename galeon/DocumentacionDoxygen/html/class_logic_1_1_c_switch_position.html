Descripción detallada

Este componente controla el movimiento de una entidad que tiene dos estados (posiciones) y la mueve de uno a otro.

Las posiciones entre las que se transita pueden configurarse desde el mapa. De la misma manera se puede estableces el atributo "loop" que mantendrï¿œ constante las transiciones. El atributo "speed" determina la velocidad del movimiento.

Recibe mensajes de tipo "SWITCH" para transitar de un estado a otro en caso de que no se encuentre en loop. El atributo "_int" del mensaje indica el estado al que se debe mover la entidad.
