Descripción detallada

Clase C3DApplication, que contiene el control de la aplicación del juego.

Extiende CBaseApplication y su máquina de estados para realizar las tareas de pintado de la escena usando Ogre. Para la interfaz de usuario utiliza OIS y CeGUI.
