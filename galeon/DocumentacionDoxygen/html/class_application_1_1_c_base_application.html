Descripción detallada

Clase CBaseApplication, que contiene el control de la aplicación del juego, implementando una máquina de estados de aplicación.

Las clases que heredan de esta se preocuparán de inicializar los subsistemas necesarios para esa aplicación concreta (subsistema gráfico, etc.), así como de la creación de los estados que componen la aplicación.

Comentarios
Las clases que implementen está clase deben inicializar el reloj (_clock).

void Application::CBaseApplication::run ( )
virtual

Función de ejecución de la aplicación.

Aquí se encuentra la ejecución del bucle principal, que se repite hasta que alguien solicite su terminación. En el bucle simplemente se comprueba si hay que realizar una transición de estado, si es así se hace efectivo el cambio. Tras estó se hace el tick() para la actualización de la vuelta.
