Lista de 'namespaces'
Lista de toda la documentación de los 'namespaces', con una breve descripción:
[detalle nivel 12]
 NApplicationNamespace con todo lo relacionado con la aplicación: clase abstracta, estados, etc
 NBaseSubsystemsNamespace con todo lo que rodea la aplicación
 NGraphicsNamespace con todo lo que se refiere a la parte gráfica
 ▼NGUINamespace con todo lo que se refiere a la interfaz con el usuario
 NButtonNamespace que contiene el tipo que define los diferentes códigos de los botones del ratón
 NKeyNamespace que contiene el tipo que define los diferentes códigos de las teclas
 NModifierNamespace que contiene el tipo que define los diferentes códigos de las teclas "modificadoras" Shift, Ctrl y Alt
 ▼NLogicNamespace que engloba la lógica del juego
 NTMessageNamespace para los tipos de mensajes posibles
 NMapNamespace con todo lo relacionado con el parseado de mapas
 NMathNamespace en el que ofrecemos alguna definición de constante matamática y métodos para convertir grados en radianes, etc
 NOgreSi se define la siguiente directiva, en modo ventana el juego no toma ratón y teclado como como de uso exclusivo sino que se sigue viendo el del OS y se pueden usar en otras ventanas
