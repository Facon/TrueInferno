Descripción detallada

Gestor de errores para PhysX.

Cuando PhysX detecta que se intenta hacer algo incorrecto (por ejemplo, llamadas a funciones con parámetros no válidos), invoca a este gestor que a su vez imprime el mensaje en la consola.
