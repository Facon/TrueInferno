bool Logic::CBuildingManager::loadInitialBuildings ( CMap *  map)

Lee y genera la lista inicial de edificios del mapa.

Aquí el resto de entidades iniciales del mapa ya han sido cargadas de fichero (map.txt) y creadas. Entre ellas, habrá diferentes Map::CEntity, cada una con un tipo específico, que servirán como "prefab" para instanciar los diferentes tipos de edificio.

Parámetros
mapMapa en el que generar los edificios.
