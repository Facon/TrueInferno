Descripción detallada

Servidor del módulo GUI que hace de interfaz con el usuario.

Está implementado como un singlenton de inicialización explícita. Sirve para comunicar a CEGUI los eventos producidos ya así hacer funcionar la interfaz gráfica de usuario y permite que se establezca que entidad es controlada por el GUI::CPlayerController. Este control se hace a través del componente Logic::CAvatarController.
