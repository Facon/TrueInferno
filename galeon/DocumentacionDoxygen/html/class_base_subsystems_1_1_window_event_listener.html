Clase auxiliar que hereda de Ogre y que sirve para modificar el comportamiento habitual de cierre de ventana.

No queremos cerrarla desde aquí, queremos que sea la aplicación.

Documentación de las funciones miembro

bool BaseSubsystems::WindowEventListener::windowClosing ( Ogre::RenderWindow *  renderWindow)
inline

En vez de dejar que el método de salida habitual funcione lo reescribimos para que se le comunique a la aplicación.

Al devolver false la ventana no se cierra, lo haremos desde la aplicación.

Este método es llamado al solicitar el cierre de la ventana tanto con Alt-F4 como al pulsar la 'x' de la ventana.

Parámetros
renderWindowVentana sobre la que se produce el evento.
Devuelve
Siempre falso, así no se realiza ninguna acción.
